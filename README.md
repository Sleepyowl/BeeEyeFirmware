
# Firmware for BeeEye Station (Environmental Sensor Data Gathering Station)

This firmware is designed for BeeEye Board hardware ([KiCad project](https://github.com/Sleepyowl/BeeEyeBoard)).

## Features

Receives data from 1Wire and BLE sensors and forwards it to the gateway station using LoRa.

Work in progress.

## Building

1. Install the **nRF Connect SDK**
   (Recommended: use the nRF Connect extension in Visual Studio Code)

2. Create a build (use `prj-release.conf` for release)

3. Flash the device
